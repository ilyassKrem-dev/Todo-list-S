### Start the website

1. Add a MONGO_URL from [mongoDB](https://www.mongodb.com/) to the `.env` file
2. Add your own JWT_SECRET or generated one to the `.env` file
3. Clone the repo
   ```sh
   gh repo clone ilyassKrem-dev/Todo-list-S
   ```
   
4. Run
   ```sh
   npm install
   npm start
   ```
